1. Клонировать с бота с гита
2. Создать на сервере файл .env в нем создать переменные среды: POSTGRES_DB, POSTGRES_USER, POSTGRES_PASSWORD NODE_TOKEN(токен бота)
3. sudo docker build -t bot . - сбилдить образ
4. sudo docker compose up
5. Подождать 2 - 3 минуты, пока запишутся все доступные ноды, затем перезапустить контейнеры(это необходимо, что бы комманда \get_nodes в боте верное количество нод)
